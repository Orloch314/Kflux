⚡ Kflux Info

Version: 1.0.0

Author: Roberto Riva

**Kflux** is an open-source desktop app to orchestrate your KNIME workflows with ease. It provides scheduling, logging, and execution history via a modern UI built with Electron and React.

## 🚀 Features


- 🧩 Manage multiple KNIME workflows

- 🕒 Per-workflow scheduler (e.g. daily at 08:00)

- 📜 Logs and execution history tracking

- 📜 Logs and execution history tracking

- 🔘 One-click control: Run / Pause / Stop

- 🧠 Live status indicators (Idle, Running, Paused, Error)

- 🖥️ Native desktop experience via **Electron**


See the README for setup details: https://github.com/Orloch314/Kflux
